# Jdroid Gradle Java Plugin
Gradle Plugin for Java Libraries

### Setup

Add the following lines to your `build.gradle`:

    buildscript {
      
      repositories {
        jcenter()
      }
      
      dependencies {
        classpath 'com.jdroidtools:jdroid-gradle-java-plugin:X.Y.Z'
      }
    }
    
    apply plugin: 'com.jdroid.java.library'

Replace the X.Y.Z by the [latest jdroid gradle plugin version](https://github.com/maxirosson/jdroid-gradle-plugin/releases/latest)
